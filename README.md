# Robust Control of the Lateral Dynamics of a Multirotor UAV

**Authors:** Crippa M., Di Mauro M.  
**Institution:** Politecnico di Milano  
**Course:** Aerospace Control Systems  
**Professor:** Marco Lovera  
**Academic Year:** 2024/2025  

---

## Overview

This MATLAB project focuses on the design and verification of a robust single-axis attitude control system for the ANT-X quadrotor UAV. The work involves grey-box modeling from flight test data, dynamic inversion control architecture implementation, and comprehensive robustness analysis.

---

## Main Topics

- Grey-box modeling of quadrotor lateral dynamics  
- Stability and control derivatives identification  
- Dynamic inversion control architecture design  
- PID controller for roll rate (p)  
- P controller for roll angle (φ)  
- H∞ synthesis with performance weights (Wp, Wq)  
- Nominal performance verification  
- Control effort limitation analysis  
- Parametric uncertainty modeling (±3σ bounds)  
- Robust stability analysis (M-Δ form, μ-analysis)  
- Robust performance verification  
- Monte Carlo validation
